About the AHA COVID19 CVD Registry Data

The COVID19 CVD Registry is a voluntary quality improvement registry, powered by AHA's Get With the Guidelines.This data was collected from 122 hospitals and is comprised of 8920 records and 431 measures. The registry entry criteria instruct hospitals to enter all patients >=18 years old who are hospitalized with an active COVID-19 infection.

Root Directory and COVID-19 CVD Data Location

  • The COVID-19 Registry Data is located in /mnt/workspace/GWTG/COVID-19. Information and documentation regarding the registry data, quality checks, data usage, as well as guidelines for analysis and publication are provided in this location workspace as well as on this site.

Software

  • The root path in the workspace is /mnt/workspace. This is the default directory for the Jupyter Home Page JupyterLab, R Studio and SAS Studio.
  • The Transfer Data Section of the PMP User-Guide provides information regarding uploading files to the workspace or getting files out of the workspace.
  • The workspace is fully customizable and tools such as MATLAB are available upon request. R Studio, Jupyter Notebook, JupyterLab, and SAS Studio are available as tabs when a workspace is launched.
  • The terminal and a variety of languages including R, Python, bash, and SAS can be accessed through Jupyter and JupyterLab. A comprehensive list of pre-installed packages and software can be found on the About Workspaces page.
